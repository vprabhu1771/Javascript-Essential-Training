`main.js` - Addition and Assignment

```
var x = 2;

x += 2;

console.log(x);
```

```
output
4
```

`main.js` - Subtraction and Assignment

```
var x = 10;

x -= 2;

console.log(x);
```

```
output
8
```

`main.js` - Multiplication and Assignment

```
var x = 10;

x *= 2;

console.log(x);
```

```
output
20
```

`main.js` - Division and Assignment

```
var x = 7;

x /= 3;

console.log(x);
```

```
output
2.3333333333333335
```

`main.js` - Modulus and Assignment

```
var x = 7;

x %= 3;

console.log(x);
```

```
output
1
```
