# 5 - JavaScript Relational Operators

1. **Less Than**

Perform comparison on both values

```
var x = 1

var y = 10

console.log(“x = “ + x);

console.log(“y = “ + y);

console.log(x + “ < “ + y);

console.log(x<y);
```

```
output

x = 1
y = 10
1 < 10
true
```
